# atostat

A modular multichannel potentiostat with bipotentiostat capabilities, designed and built with [atopile](https://atopile.io).

## Specifications

| Parameter      | Value                                        |
| -------------- | -------------------------------------------- |
| Current range  | 10 nA to 1 A per channel                     |
| Voltage range  | +-2 V                                        |
| Bipotentiostat | Yes                                          |
| Architecture   | Modular per-channel                          |
| Connectivity   | Ethernet, USB, WiFi (per-channel addressable)|
| Software       | Scriptable API + polished GUI                |
